# TaskManagement Frontend

==========================

## Tech stack used
- React with TypeScript
- TailwindCSS

## How to run the project
### 1. Clone The Repo

`git clone "https://github.com/Priyanshu629/taskManagementFrontend"`

### 2 . Install dependencies

 `npm install`

### 3 . Run the project

`npm run dev`
